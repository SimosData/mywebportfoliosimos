Desktop File Explorer & Sorter

Project Description

This is a desktop application built with Python and Tkinter. It functions as a file explorer, allowing users to navigate their file system. Additionally, it includes a feature to sort files within a selected directory into different folders based on their file type (e.g., images, documents, videos).

Technologies Used

  • Python
  • Tkinter (Python's standard GUI library)
